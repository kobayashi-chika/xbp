注釈付きのソースコードはGithub ( Gitlabにミラーリング) で入手でき、 MIT ライセンスの下でリリースされています。誰でもこれらの色を何にでも使用できます。

合理的な色はアクセシブル.

カラー システムは、アクセスしやすく、コントラストの高い色の組み合わせを作成するために構築されました。

WCAG 2 コントラストの推奨事項と色の色合い

Reasonable Colors を使用してWCAG 2 のコントラストに関する推奨事項を満たすのは簡単です。各色には、番号が付けられた 6 つの色合いがあります。2 つのシェード間のコントラストは、シェード番号の違いによって推測できます。

MDN でアクセシブルな色のコントラストの詳細をお読みください。

コントラスト比 ≥ 3:1 : 陰影差 2

差が 2 のシェード間のコントラスト (たとえば、シェード 2 とシェード 4) は、少なくとも 3:1 になります。

✓ 大きなテキストの
WCAG 2 AA コントラスト ✓ アイコンとグラフィックスの WCAG 2 AA コントラスト

コントラスト比 ≥ 4.5:1 : 陰影差 3

差が 3 のシェード間のコントラスト (たとえば、シェード 2 とシェード 5) は、少なくとも 4.5:1 になります。

✓ 本文の WCAG 2 AA コントラスト
✓ 大きいテキストの WCAG 2 AAA コントラスト

コントラスト比 ≥ 7:1 : 陰影差 4

差が 4 のシェード間のコントラスト (たとえば、シェード 2 とシェード 6) は、少なくとも 7:1 になります。

✓ 本文の WCAG 2 AAA コントラスト

楽しい部分

そしてこれが本当に楽しい部分– これはすべて LCH 色空間内に構築されており、各色合いの相対輝度が特定の範囲内に固定されているため、コントラスト ルールは 24 色セットすべてで機能します. グレーも含めて、あらゆる色の色合いをミックスしてマッチさせます。

注:純白 (#FFFFFF) はすべてのシェード 1 よりも相対輝度が高く、純黒 (#000000) はすべてのシェード 6 よりも相対輝度が低くなります。したがって、純粋な白はシェード 1 の色として使用でき、純粋な黒はシェード 6 の色として使用できます。すべてのコントラスト比の規則が適用されます。

合理的な色は使いやすい.

24 セットの名前付きの色は等間隔で配置され、各色相は LCH カラー ホイール上の隣の色相から 15° 離れています。この均一な間隔により、補完的なペアを簡単に見つけて見栄えの良いカラー パレットを作成できます。

注:各色の詳細ページ (例: indigo ) には、いくつかの単純で科学的な色の組み合わせが含まれています。

入門

Reasonable Colors は、次の形式に従う一連の CSS 変数を定義します。

--color-COLORNAME-SHADE /* Example */ code { background-color: var(--color-magenta-2);

開始する最も簡単な方法は、単一の CSS ファイルを含めることです。

<link rel="stylesheet" href="https://unpkg.com/reasonable-colors@0.4.0/reasonable-colors.css">

または、NPM 経由でインストールします。

npm install reasonable-colors

その他の色空間

簡単にするために、デフォルトの Reasonable Colors CSS 変数は 16 進値を使用して色を定義します。ただし、HSL と RGB の両方の CSS 変数も使用できます。

<link rel="stylesheet" href="https://unpkg.com/reasonable-colors@0.4.0/reasonable-colors-hsl.css"> <link rel="stylesheet" href="https://unpkg.com/reasonable-colors@0.4.0/reasonable-colors-rgb.css">

LCH カラーのサポート

近い将来、Web ブラウザーが広色域の色空間をサポートし始める予定です。Reasonable Colors は LCH 色空間を使用して構築されているため、この日の準備は整っており、すでに LCH CSS 変数を提供しています。

<link rel="stylesheet" href="https://unpkg.com/reasonable-colors@0.4.0/reasonable-colors-lch.css">

SCSS 変数

対応する .scss ファイル (色空間ごと) は、SCSS 変数宣言と共に提供されます。
